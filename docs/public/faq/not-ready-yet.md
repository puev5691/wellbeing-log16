# FAQ: Что пока не готово?

## Краткий ответ

log16 уже полезен локально, но ещё не является готовым публичным продуктом.

## Пока не готово

    hosted demo
    public production release
    full install guide
    complete FAQ
    verified Windows/WSL test bench
    full contribution workflow
    public GitHub publication
    stable external support process

## Почему это важно говорить честно

Публичный вход должен не обещать больше, чем система может выдержать.

Лучше назвать ограничение заранее, чем потом чинить ожидания людей ломом и извинениями.
