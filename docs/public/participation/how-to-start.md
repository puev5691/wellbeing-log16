# Как начать участие

## Краткий ответ

Начать можно не с установки всего проекта, а с выбора уровня участия.

## Уровни входа

### Уровень 0. Читатель

    прочитать public docs
    понять проект
    задать уточняющий вопрос

### Уровень 1. GitHub user

    читать docs
    смотреть issues/tasks
    предлагать правки к текстам
    уточнять FAQ

### Уровень 2. Local AI tester

    установить Ollama
    скачать qwen3:8b
    запустить log16 doctor
    прислать diagnostic report

### Уровень 3. Developer

    читать код
    запускать tests
    делать small patches
    улучшать CLI, doctor, dashboard, docs

### Уровень 4. Infrastructure contributor

    помогать с hosted demo
    помогать с deployment
    помогать с monitoring
    помогать с reproducibility

## Главный принцип

Не нужно начинать с максимального уровня.

Минимальный полезный вклад тоже считается вкладом.
