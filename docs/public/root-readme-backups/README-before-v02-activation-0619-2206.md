# wellbeing-log16

## Краткий смысл

`wellbeing-log16` — программа для работы с информационным полем проекта БЛАГОПОЛУЧИЕ:

- вопросы;
- темы;
- задачи;
- Сущности;
- ответы Сущностей;
- review;
- решения ОПЕРАТОРА;
- подготовка канонов и reusable answers.

## Статус

`0.1.0-lab`

Это лабораторная сборка. Код только начал отделяться от runtime state.

## Текущая архитектура

Код программы собирается в этом репозитории:

    /data/wellbeing/repos/wellbeing-log16

Живое состояние системы должно находиться отдельно:

    /data/wellbeing/obs/log16

Старый runtime пока остаётся здесь:

    /data/wellbeing/obs/consultant/outbox/log16-kernel

## Что сейчас в репозитории

- `src/log16/` — будущий чистый пакет программы.
- `src/log16_legacy/bin/` — текущие рабочие скрипты, скопированные из старого runtime.
- `schemas/` — JSON schemas.
- `docs/` — документы по сборке, миграции и архитектуре.
- `scripts/` — служебные скрипты для подготовки и будущей миграции.

## Что нельзя коммитить

Нельзя публиковать в GitHub:

- runtime state;
- ответы Сущностей;
- приватные документы;
- рабочие архивы;
- API keys;
- secrets;
- персональные данные участников;
- сырые logs.
