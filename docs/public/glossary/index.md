# Glossary

## Краткий смысл

Этот раздел собирает термины проекта, чтобы внешний читатель не тонул в наших внутренних названиях.

## Начальные термины

### БЛАГОПОЛУЧИЕ

Проект построения системы совместной деятельности людей, программных инструментов и AI-сущностей.

### log16

Локальный экспериментальный контур работы с вопросами, ответами, задачами, review и reusable answers.

### reusable answer

Ответ, который прошёл проверку и может использоваться повторно в похожих ситуациях.

### doctor

Диагностическая команда, которая проверяет окружение log16 и возвращает человекочитаемый или JSON-отчёт.

### runtime

Рабочая среда выполнения log16, где лежат очереди, ответы, review и отчёты.

### public knowledge base

Публичная база ответов и объяснений проекта.
