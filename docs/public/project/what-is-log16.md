# Что такое log16

## Краткий ответ

log16 — это экспериментальный локальный контур для работы с вопросами, ответами, задачами, review и повторно используемыми знаниями проекта.

## Что он делает

log16 помогает организовать путь:

    входящий вопрос
    тема
    задача
    маршрут к сущности
    ответ
    review
    approved/reusable answer
    новая задача или обновление знания

## Что уже есть

    local runtime
    CLI
    pult
    dashboard
    doctor diagnostics
    JSON doctor report
    productization docs
    Windows/WSL onboarding draft

## Что ещё не готово

    публичный hosted demo
    полноценная база готовых ответов
    внешний install guide
    Windows/WSL проверенный стенд
    публичная GitHub-публикация

## Главная граница

log16 пока является local lab system.

Публичное использование требует отдельной подготовки.
