Публичный GitHub repo не должен обещать мгновенный запуск всем подряд.

Repo должен честно разделять уровни входа:

    читатель
    GitHub-пользователь
    локальный AI-тестировщик
    разработчик
    инфраструктурщик
    участник проекта
