# Open task classes

## Краткий смысл

Этот документ описывает не конкретные issues, а классы задач, которые могут быть понятны внешнему участнику.

## Documentation tasks

    улучшить FAQ
    уточнить glossary
    переписать сложный текст проще
    добавить example answer cards

## Testing tasks

    запустить doctor
    проверить local AI setup
    проверить Windows/WSL route
    проверить dashboard route
    прислать structured report

## Development tasks

    улучшить CLI
    улучшить doctor JSON
    добавить export report
    улучшить tests
    добавить examples

## Productization tasks

    собрать public README
    собрать local AI onboarding
    собрать hosted demo requirements
    описать contribution path

## Review tasks

    проверить публичные тексты
    найти неясные места
    отметить outdated sections
    предложить better answer structure
