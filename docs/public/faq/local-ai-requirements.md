# FAQ: Можно ли запустить log16 локально?

## Краткий ответ

Да, но полноценный локальный запуск требует подготовленного окружения.

## Минимальная рабочая логика

Для текущего local lab route нужны:

    Linux-like окружение
    bash
    git
    Python
    Ollama
    qwen3:8b
    log16 runtime
    умение запускать diagnostics

## Проверка

Основная команда диагностики:

    log16 doctor
    log16 doctor --json

## Важная граница

Без Ollama и qwen3:8b можно изучать документацию и часть структуры, но полноценная работа local AI контура будет ограничена.
