# Public knowledge base

## Краткий смысл

Этот раздел предназначен для публичного объяснения проекта БЛАГОПОЛУЧИЕ, log16 и возможных маршрутов участия.

Это не runtime.

Это не внутренний архив.

Это не полный проектный канон.

Это публичная входная база, которую можно безопасно развивать перед будущей публикацией.

Публичная база должна отвечать на вопросы так, чтобы человек мог:

    понять проект
    понять текущую стадию
    увидеть ограничения
    выбрать уровень участия
    перейти к конкретным задачам
