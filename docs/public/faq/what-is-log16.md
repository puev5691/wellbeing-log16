# FAQ: Что такое log16?

## Краткий ответ

log16 — это экспериментальный локальный контур проекта БЛАГОПОЛУЧИЕ для работы с вопросами, ответами, задачами, review и reusable answers.

## Развёрнутый ответ

log16 помогает организовать путь:

    входящий вопрос
    тема
    задача
    маршрут к сущности
    ответ
    review
    reusable answer
    новая задача или обновление знания

На текущем этапе log16 является local lab system.

Он уже имеет CLI, dashboard, pult, doctor diagnostics и JSON doctor report, но пока не является массовым публичным продуктом.
