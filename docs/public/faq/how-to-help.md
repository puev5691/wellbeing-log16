# FAQ: Как помочь проекту?

## Краткий ответ

Можно помогать на разных уровнях: от чтения и уточнения FAQ до разработки, тестирования и инфраструктуры.

## Уровни участия

    читать public docs
    улучшать формулировки
    проверять FAQ
    запускать doctor
    тестировать local AI setup
    помогать с Windows/WSL route
    писать code patches
    улучшать docs
    описывать ошибки структурированно

## Минимальный полезный вклад

Минимальный полезный вклад — это не обязательно код.

Полезным может быть:

    найденная неясность
    уточнённый вопрос
    исправленная формулировка
    диагностический отчёт
    проверка инструкции
