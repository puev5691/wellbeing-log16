# FAQ: Можно ли использовать Windows?

## Краткий ответ

Windows support рассматривается как экспериментальный маршрут через WSL2.

## Основной кандидатный путь

    Windows 11
    WSL2
    Ubuntu
    Ollama
    qwen3:8b
    log16 repo/runtime внутри Linux-like layout

## Что пока не подтверждено

    полный Windows/WSL test bench
    нативный PowerShell-only route
    hosted demo
    внешний install guide

## Правильное ожидание

Windows users смогут участвовать, но этот маршрут требует отдельной подготовки и тестирования.
